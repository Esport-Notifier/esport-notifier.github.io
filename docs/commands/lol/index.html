League of Legends

Help

Command : .en lol help

Matches

The commands that are in this table start with .en lol matches.

Available options :

Name Description
past Show the results of the matches of the past seven days
running Show the score of ongoing matches
upcoming Show the schedule of matches for the next two days

Events

The commands that are in this table start with .en lol events.

Available options :

Name Description
past Show the results of the matches of the past seven days
running Show the match score of ongoing matches
upcoming Show the schedule of matches for the next two days
